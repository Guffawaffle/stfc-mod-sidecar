STFC Mod Sidecar

Companion Console

A local-first companion surface for diagnostics, settings, event review, and future install/update workflows.

Boundary

Outside The Game Process

The sidecar reads local files and documented local APIs. It does not inject into STFC, synthesize input, automate gameplay, or send arbitrary commands to the mod.

Configuration

Schema First

The companion UI should render settings from the shared schema, persist sparse startup config, and live-apply only schema-approved runtime settings.

Packaging

Optional Installer

Manual DLL/dylib installs stay available. The companion installer can add update, launch, diagnostics, and local viewer workflows for users who want them.

Visual Credit

LCARS Inspired

This interface uses original project-authored styling inspired by LCARS interface language. It does not bundle TheLCARS.com template files.
