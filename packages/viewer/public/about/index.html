STFC Mod Sidecar

Companion Console

A local-first companion surface for diagnostics, settings, event review, and future install/update workflows.

Boundary

Outside The Game Process

The sidecar reads local files and documented local APIs. It does not inject into STFC, synthesize input, automate gameplay, or send arbitrary commands to the mod.

Security

Security Is Paramount

Desktop bootstrap accepts only a canonical local game directory with prime.exe directly inside it before the sidecar reads or writes derived mod files.
